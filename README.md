# Probot-Express

A mashup of [Express](https://expressjs.com/) and
[Probot](https://github.com/probot/probot), starting point of an Express
template for "full webapp" GitHub Apps using Probot.

## Setup

```sh
# Install dependencies
npm install

# Run the bot
npm start
```
